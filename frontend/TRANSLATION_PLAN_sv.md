## Translation Guidelines

### General Principles
1. **Maintain Structure:** Keep all code structure, formatting, and TypeScript types unchanged
2. **Professional Tone:** Use formal Swedish appropriate for official municipal communications
3. **Consistency:** Maintain consistent terminology throughout (create a glossary if needed)
4. **Context Awareness:** Consider the UI context when translating
5. **Gender Neutrality:** Use gender-neutral Swedish where appropriate

### Key Translation Considerations
- **Reference Translations:** Check existing Finnish and Swedish translations in `citizen/i18n/fi.tsx` and `citizen/i18n/sv.tsx` for terminology examples and consistency
- **Finnish ‚Üí Swedish:** Not just direct translation; consider Swedish conventions
- **Municipality Terms:** Use proper Swedish terms for administrative/educational contexts
- **Date Formats:** Ensure date-related strings are culturally appropriate
- **Legal Terms:** Pay special attention to legal/official terminology
- **Abbreviations:** Update any abbreviations to Swedish equivalents

### Technical Constraints
- **Preserve:** All variable interpolations like `${variable}`
- **Preserve:** All JSX/React components and their props
- **Preserve:** All function signatures and TypeScript types
- **Test:** After each section, verify no TypeScript errors introduced
